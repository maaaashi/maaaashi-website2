# Development

```bash
$ git clone https://github.com/maaaashi/maaaashi-website2.git
$ cd maaaashi-website2
$ npm install
```

# Start Server

```bash
$ make dev
```

Access to http://localhost:4321

# E2E

$ cd e2e/
$ gauge run specs
