# Development

```bash
$ git clone https://github.com/maaaashi/maaaashi-website2.git
```

## Start Server

```bash
$ cd maaaashi-website2/web
$ npm install
$ npm run dev
```

Access to http://localhost:4321

## E2E

```bash
$ cd e2e/
$ npm install
$ npm run test
```
